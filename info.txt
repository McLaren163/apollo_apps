Параметры выбора чертежа/sketch:
    - модель ворот (выбираем таблицу типов)
    - тип рамы (выбираем тип из таблицы типов)
    - сторона отката
    - тип консоли

{
    gate_type: prestige
    size_type: a
    side_type: left
    console_type: triangle
    filename: path to file
}

Валидациия введенных данных:
    + место отката / размер консоли
    + количество зубчатой рейки
    + целое число
    - рама от типа ворот
    - рама от типа заполнения
    - тип консоли от ширины проема (ошибка выбора четрежа)
